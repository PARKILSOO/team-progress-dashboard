# Claude Code Configuration

## Agent skills

### Issue tracker

Issues are tracked locally as markdown files in `.scratch/` — no GitHub/GitLab integration. See `docs/agents/issue-tracker.md`.

### Triage labels

Standard five-label triage vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout: `CONTEXT.md` and `docs/adr/` at the repo root. See `docs/agents/domain.md`.
